jQuery fullscreenImage plugin.

Just a simple plugin that makes any image you have a fullscreen background image without breaking the aspect ratio of your image.

How To:

To use simply create a container with an image inside then pass this container into the fullscreenimage() jquery plugin.

Options:

There are some basic controls you should be aware of. For a list of options and their defaults check out the code box below.

Plugin options and the defaults: maxUpscale: 1 (upscale ratio, eg: 1.5 would upscale the image to 150%, then use the rasterImg overlay) rasterImg: '/static/img/raster.png' (a 2px raster that neatly disguises the bad effects of image upscaling) showRaster: true | false (easily disable the raster all together) vAlign: 'center' | 'top' | 'bottom' (set the anchor point of the image's vertical axis)

Compatibility

This little plugin has been tested in Internet Explorer 6+, Opera 10.6, Firefox 3.6+, Google Chrome & Safari 5.
